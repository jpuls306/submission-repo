Lists: A lot of web content is lists and HTML has special elements for these. Marking up lists always consists of at least two elements. The most common list types are ordered and unordered list "<ul>" and "<ol>". Unordered list are for lists where the order of the items does not matter, such as a shopping list. These are wrapped by the element shown above/before ul. Ordered lists are for lists where the order of the items does matter, such as a list of cooking instructions in a recipe. These are wrapped by the ol element. Each item inside the lists is put inside an "<li>" element which stands for list item.

What is CSS?

Like HTML, CSS is not a programming language. It's not a markup language either. CSS is a style sheet language. CSS is used to style HTML elements: you select the elements you want to style and set values for their style properties, which define how they will look.
